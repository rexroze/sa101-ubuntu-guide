# SA101 — Installing Ubuntu Desktop in VirtualBox

A step-by-step guide for installing Ubuntu Desktop in Oracle VirtualBox, made for System Administration class (SA101).

## Live Guide

**https://rexroze.github.io/sa101-ubuntu-guide/**

## What's Inside

- **Installation Guide** — Full walkthrough from pre-flight checks through snapshot
- **Report Example** — Sample OS Installation Report for Part 17 submission

## How to Use

1. Open the [live guide](https://rexroze.github.io/sa101-ubuntu-guide/)
2. Follow each step in order
3. Use the report example as reference when writing your own

## Requirements

- Windows 10/11 (64-bit)
- 8 GB+ RAM recommended
- 40–50 GB free storage
- VirtualBox 7.2+
- Ubuntu Desktop ISO
